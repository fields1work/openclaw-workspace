## 🚀 HOW TO USE (Right Now)

### Option 1: Command Line (Quickest)

```bash
# Install
pip install edge-tts

# Generate TTS
edge-tts --voice en-US-GuyNeural --rate=+10% --text "My wife was living a double life" --write-media output.mp3 --write-subtitles output.srt

# Listen
# (Open output.mp3 in any media player)
```

### Option 2: Python Integration

```python
import asyncio
from automation.edge_tts_generator import EdgeTTSGenerator

# Initialize
tts = EdgeTTSGenerator()

# Generate TTS
asyncio.run(tts.generate_tts(
    text="My wife was living a double life",
    output_path="output.mp3",
    voice="en-US-GuyNeural",
    rate="+10%",
    write_subtitles=True
))
```

### Option 3: Full Automation

```bash
# Generate 3 complete video packages
python automation.py --full 3

# (Now uses Edge-TTS instead of ElevenLabs)
```
