## 🥇 OPTION 1: CAPCUT (RECOMMENDED - FREE)

**Why CapCut:** Free, TikTok-optimized, best text animations, easy captions

### PERFECT EXPORT SETTINGS:

**Step-by-Step:**

1. **File > Export** (or top right Export button)
2. **Resolution:** 1080×1920 (Full HD)
   - ❌ NOT 720p
   - ❌ NOT "Auto"
   - ✅ 1080p minimum

3. **Frame Rate:** 60fps
   - For gameplay content = smoother
   - For story content = acceptable at 30
   - **Recommendation:** 60fps always

4. **Bitrate:** HIGH
   - **Mobile:** Tap "More" > "Bitrate" > Select "High"
   - **Desktop:** Look for Quality slider → Drag to MAXIMUM
   - **Target:** 30-50 Mbps (CapCut calls this "High")

5. **Format:** MP4 (H.264)
   - Most compatible
   - TikTok optimized
   - Smaller files than MOV

6. **Advanced Settings:**
   - **If available:** Turn OFF "Smart Compression"
   - **If available:** Turn OFF "Optimize for Web"
   - These CAUSE quality loss

### CAPCUT QUALITY TIERS:

| Setting | Bitrate | Quality | File Size | TikTok Result |
|---------|---------|---------|-----------|---------------|
| **Low** | 2-5 Mbps | ❌ Pixelated | 5-10 MB | Blurry |
| **Medium** | 8-12 Mbps | ⭐⭐ Okay | 15-25 MB | Acceptable |
| **High** | 30-50 Mbps | ⭐⭐⭐⭐ Crisp | 80-150 MB | **PERFECT** |
| **4K** | 80+ Mbps | ⭐⭐⭐⭐⭐ Overkill | 300+ MB | Wasted |

## 🔥 CAPCUT STEP-BY-STEP WORKFLOW

### PHASE 1: IMPORT (2 min)

1. **Open CapCut Desktop**
2. Import → Select:
   - `gameplay_video.mp4` (Subway Surfers/GTA/Minecraft)
   - `voiceover.mp3` (from ElevenLabs/GTTS)
   - Any images (thumbnails, etc.)

### PHASE 2: LAYOUT (3 min)

1. **Video on bottom layer** (gameplay)
2. **Text on top layers** (captions)
3. **Audio:** Voiceover track
4. **Background music:** Optional (10-15% volume)

### PHASE 3: CAPTIONS (5 min)

**Method A: CapCut Auto-Captions**
1. Select voiceover
2. Text > Auto Captions
3. Wait for transcription
4. Edit text if needed

**Method B: Manual (Better Control)**
1. Text > Add Text
2. Type first phrase
3. Style:
   - Font: **League Spartan** or **Bebas Neue**
   - Size: 48-58**
   - Color: White
   - Stroke: 3-4px Black
   - Shadow: Off or subtle
4. Position: Lower 2/3, centered
5. Duplicate for each phrase
6. Sync timing to voiceover

### PHASE 4: HOOK (1 min)

1. First 3 seconds = CRITICAL
2. Biggest text, brightest color (Orange #ff6b35)
3. Animation: "Typewriter" or "Pop"
4. Text: First hook line from script

### PHASE 5: EXPORT (2 min)

1. **Export button** (top right)
2. **Settings:**
   - Resolution: 1080×1920
   - Frame Rate: 60fps
   - Bitrate: **HIGH** (tap "More" on mobile)
3. **Export**
4. **Save to:** Phone gallery or laptop
