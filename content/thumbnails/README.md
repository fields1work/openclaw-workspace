## How To Use

### Option 1: Convert to PNG (Recommended)
Use an online converter or tool:
1. Go to a site like cloudconvert.com or convertio.co
2. Upload the .svg file
3. Download as PNG (1080x1920)
4. Upload to TikTok as custom thumbnail

### Option 2: Use as Reference
Open the SVG in browser (double-click it) and screenshot, or use as a guide to recreate in Canva.

### Option 3: Direct Import
Some editing apps (like Inkscape, Adobe Illustrator, or even PowerPoint) can open SVG files and export high-res images.
