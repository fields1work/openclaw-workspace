## PHASE 4: SET PERMISSIONS (15 min)

### Make #welcome, #rules, #roadmap Read-Only:
1. Right-click channel → "Edit Channel"
2. Click "Permissions" tab
3. Click "+" next to @everyone
4. Set:
   - View Channel: ✅
   - Send Messages: ❌ (click X)
   - Read Message History: ✅
5. Click "Save Changes"

### Make #announcements Admin-Only:
1. Same process as above
2. Set @everyone Send Messages: ❌
3. Add your role with Send Messages: ✅
