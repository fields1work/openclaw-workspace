# Content Pipeline

## Folders

- **ideas/** → Raw content ideas, hooks, research
- **scripts/** → Written scripts for videos (Reddit stories, facts, etc.)
- **voiceovers/** → ElevenLabs audio files
- **ready/** → Finished videos ready to post
- **posted/** → Archive of posted content + performance metrics

---

## Workflow

1. **Ideas** → Brainstorm in `ideas/`
2. **Scripts** → Write in `scripts/YY-MM-DD-title.md`
3. **Voice** → Generate in ElevenLabs → Save to `voiceovers/`
4. **Edit** → CapCut → Export to `ready/`
5. **Post** → TikTok → Move to `posted/`
6. **Track** → Log performance in Mission Control
