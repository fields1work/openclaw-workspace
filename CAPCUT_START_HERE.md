## 📋 QUICK START GUIDE

### Step 1: Download CapCut Desktop
- Go to: www.capcut.com
- Download for Windows
- Install (free)

### Step 2: Open CapCut
- Click "New Project"
- Click "Import"
- Select gameplay video + voiceover

### Step 3: Follow Breakdown
- Open `CAPCUT_COMPLETE_BREAKDOWN.md`
- Copy text line-by-line
- Paste into CapCut text boxes
- Match timing to voiceover

### Step 4: Style Everything
- Hook (first 3 sec): Orange #ff6b35, 64pt
- Body: White, 54pt, black stroke
- Key words: Yellow #ffcc00, 58pt

### Step 5: Export
- Settings: 1080×1920, 60fps, HIGH bitrate
- Save to `content/capcut_ready/video_XXX/`
