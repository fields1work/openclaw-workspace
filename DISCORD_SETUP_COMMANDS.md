# DISCORD SETUP COMMAND LIST

## To complete Discord setup, run these commands in order:

### Step 1: Create Text Channels
1. Click the "+" next to "Text Channels"
2. Create these channels (copy/paste names):
   - welcome
   - announcements
   - general (already exists)
   - content-showcase
   - strategy-discussion
   - resources-and-tools
   - wins-and-milestones

### Step 2: Create Categories
1. Right-click server name → "Create Category"
   - "WELCOME" (move welcome, announcements here)
   - "GENERAL" (move general here)
   - "CONTENT" (move content-showcase here)
   - "STRATEGY" (move strategy-discussion here)
   - "RESOURCES" (move resources-and-tools here)
   - "WINS" (move wins-and-milestones here)

### Step 3: Create Voice Channels
1. Click "+" next to "Voice Channels"
   - General
   - Daily Sync
   - Deep Work
   - AMA Sessions

### Step 4: Configure Permissions
1. Click server name → "Server Settings"
2. Go to "Roles" → Create "Member" role
3. Go to "Channels" → Set category permissions
